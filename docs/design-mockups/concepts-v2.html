3가지 모두 공유하는 요소

이번엔 우열 비교가 아니라 공통 design language를 정한 후 톤만 다르게 만든 거라, 어느 걸 선택해도 baseline 토큰이 거의 같습니다:

  • Surface: paper-100 (#FFFEFB) / paper-200 (#FBFAF7) · 차갑지 않은 paper white
  • Ink: ink-900 (#1A1614) warm charcoal — 순흑이 아님
  • Border: paper-edge (#E8E4DC) hairline · 거의 안 보일 정도
  • Type: Geist sans + Geist Mono 숫자 + (B만) Source Serif 4 헤드라인
  • Tabular nums 모든 숫자 · font-variant-numeric: tabular-nums
  • Accent: terracotta (#C2410C) 또는 aubergine (#5B2A4D) — 둘 다 indigo 보단 따뜻

선택 후 진행: 마음에 든 variant 알려주세요 (또는 "A의 카드 + B의 헤드라인 + C의 펄스 점" 같은 hybrid). 그 위에 design.md (토큰 / 컴포넌트 baseline / Tailwind config) 작성합니다.
