脳の視覚認識能力

00:05

脳は異なる解像度・スタイルの画像を同じ物体として認識できる。これは深層学習が解こうとしている根本的な問題。

Take 脳は異なる解像度の画像を同じ物体として認識できる驚異的な能力を持つ
  • 28×28 pixel の極小画像でも「3」と認識できる
  • スタイル・大きさ・解像度に依存しない汎化能力
  • この能力こそ neural network が模倣しようとしている対象
visual_recognition neural_network generalization
Check 脳がどのように低解像度の画像を認識するか、その仕組みを 30 字以内で説明せよ。

Neural Network の基本構造

02:14

Input layer → Hidden layers → Output layer の階層構造。各 layer は前 layer の出力を入力として受け取り、weighted sum + activation function を適用する。

Neuron 0〜1 の値を保持する単位。視覚認識の場合、各 pixel の輝度値から始まる。
Activation neuron の現在値。input layer では pixel 値、後段 layer では前 layer の重み付き和に sigmoid を通したもの。
