Study-Helper UI Mockups

同じデータ (持続可能性とガバナンス K-LMS テストセッション) を 4 つのレイアウト案でレンダリング. 直接比較してください.

A. Refined Compact

保守的 / 폴리시

현재 디자인의 정제판. 색상 시스템을 다듬고 ★ 시각 임팩트 강화.

  • 🔵 indigo 섹션 헤더
  • 🟡 amber 用語 카드
  • 🔴 rose ★ 박스

B. Notion-style Hierarchy

아이콘 / 깔끔

아이콘 + 들여쓰기로 정보 종류 즉시 인식. 학습 노트 도구 패러다임.

  • 📘 用語 / ⭐ 중요 / 📌 일반 / 💡 예시
  • 호버 시 timestamp (visual noise ↓)
  • 섹션은 collapsible 가능

C. Magazine Cards

시각적 / 매력

각 섹션 = 그라디언트 카드. 「試験で覚える」 박스로 학습 가치 즉시 어필.

  • 섹션마다 다른 그라디언트 (위치감)
  • ⭐ 「試験で覚える」 명시 박스
  • 用語 / 具体例 / 補足 명시 라벨

D. Live + Notes Split

실용 / 모드 전환

사용 시점에 따라 3 모드. 강의 청취 vs 복습은 다른 화면.

  • 🎧 リスニング: 자막 크게
  • 📝 復習: 노트만
  • 📊 並列: 둘 다
