# Lisna

> 1時間の音声を、5分で読める資料に — every spoken sound, structured
> on the user's own device.

Lisna turns live audio into reading-grade notes: classroom lectures,
LMS recordings, Zoom/Teams meetings, in-room conversations. v1 ships
as a Chrome extension running on AWS Lambda (student beta — Japanese
universities). v2 is the desktop, on-device native build.
